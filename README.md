# RescuePath
This repository contains a Problem-Based Learning (PBL) project developed by 4th semester students using Design and Analysis of Algorithms (DAA). The project focuses on implementing shortest-path algorithms to simulate an emergency routing system.

# 📌 Overview
RescuePath is a GUI-based web application developed using Design and Analysis of Algorithms (DAA). It simulates an emergency route finding system by computing optimal paths using graph-based shortest path algorithms under dynamic traffic conditions.

# 📚 Features

• Shortest path computation using Dijkstra algorithm
• Graph-based road network representation
• Dynamic traffic simulation
• Interactive GUI for route visualization
• Connectivity check using BFS/DFS
• Real-time route recalculation (rerouting)

# 🛠 Technologies Used

• HTML – Structure of the web application  
• CSS – Styling and UI design  
• JavaScript – Logic and interactivity  
• JSON – Data storage for roads and nodes  


# 🧠 Concepts Used

• Graph Theory (Nodes and Edges)  
• Shortest Path Algorithms (Dijkstra)  
• BFS / DFS Traversal  
• Greedy Approach  
• Time and Space Complexity

GUI-based Web Application (Academic Simulation)
